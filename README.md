This chatbot named "NIGAChatbot" communicates with its users through prepared responses. This chatbot, for instance, knows all greetings and farewells and can answer some questions to its name or the weather or even the jokes. Through the 'get_response', it checks input from the user and returns response according to words matched, while the method 'start_chat' controls the ongoing conversation because it prompts user input with responses until some farewell expression is detected while ending that chat. Hence, this simple design makes this chatbot interactive and possible to develop further.
